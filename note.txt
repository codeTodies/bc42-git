-git init: khởi tạo git cho 1 project(chỉ dùng 1 lần duy nhất)
-git add filename1 filename2: thêm 1 hoặc nhiều file bị thay đổi
-git add .:thêm tất cả file bị thay đổi
-git commit -m "message":xác nhận sự thay đổi và lưu trữ tất cả các file đã thay đổi
-git add:sắp xếp các file đã bị thay đổi, thêm mới hoặc xóa so với lần commit trước đó, chuẩn bị cho lần commit tiếp theo
-git log: xem lại lịch sử commit
- git status: xem trạng thái hiện tại của thư mục làm việc, hiển thị những thay đổi so với lần commit trước đó đồng thời hiển thị trạng thái của các file đó như là đã được xác nhận để commit hay chưa
git push:đẩy tất cả commit lên trên remote repository
-git clone: tải remote repository về máy tính
-git checkout: chuyển sang branch khác
+git checkout branch-name:chuyển branch
+git checkout -b branch-name:tạo brench
