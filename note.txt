-git init: khởi tạo git cho 1 project(chỉ dùng 1 lần duy nhất)
-git add filename1 filename2: thêm 1 hoặc nhiều file bị thay đổi
-git add .:thêm tất cả file bị thay đổi
-git commit -m "message":xác nhận sự thay đổi và lưu trữ tất cả các file đã thay đổi
-git add:sắp xếp các file đã bị thay đổi, thêm mới hoặc xóa so với lần commit trước đó, chuẩn bị cho lần commit tiếp theo
